# qtltools
A package for manipulating and plotting information from R/QTL objects generated from the package `qtl`. This was mainly intended to generate plots like that given by [Ã…gren *et al.* ](http://www.pnas.org/content/110/52/21077.short) in future papers.

## Installation
Using devtools from within R:

```
install.packages('devtools')
devtools::install_github("ellisztamas/qtltools")
```
